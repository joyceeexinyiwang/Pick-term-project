This video is about Pick!, a Django web app that allows a group of people to rate/compare/choose things together. We can use it for shopping, picking a club apparel, choosing a restaurant, and so on.

It is a term project for a intro-level programming class that I took at Carnegie Mellon. I was pretty lucky so I got to present it in front of a 400-people class. I am actually a TA for that class now.
