Pick!

"Pick!" is a Django web app that allows a group of people to rate/compare/choose things together. We can use it for shopping, picking club apparels, choosing restaurants, and so on.

This was my term project for a intro-level programming class at Carnegie Mellon. I was very lucky to have had the chance of presenting it in front of our class. I am actually a TA for that class now.
